# rplane
Raylib plane skeleton project

## Pre-requisites
- GNU/Linux x86 (built on WSL)
- cmake
- Raylib (https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux) and its dependencies

## Setup
- `mkdir build && cd build`
- `cmake ..`
- `cmake --build .`

## Run
- `./rplane`
